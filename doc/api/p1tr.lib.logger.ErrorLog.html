Class p.l.l.ErrorLog:

Part of p1tr.lib.logger
Error log for collecting information about problems which occur while running P1tr. level specifies the lowest kind of error which is shown inside the console and must be one of those: n, f, e, w, i, where n stands for nothing and i for everything. Default loglevel is f.
